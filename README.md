### Create a Kotlin project /w Gradle

#### `cd /D <project path>`

#### `gradle init`

#### Then press 2 and 4 in first two questions (Application & Kotlin)

### Run a Kotlin project via Gradle in CLI

#### Run the app: `./gradlew run`

#### Build the app: `./gradlew build`

#### `./gradlew build --scan`
